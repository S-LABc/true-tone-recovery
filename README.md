# true-tone-recovery
Чтение и запись памяти дисплея от iPhone. Основан на стандартной библиотеке EEPROM.h

# Назначение файлов

* Sources - Исходники для Arduino и C#
* Support - Вспомогательные файлы
* Apple True Tone Recovery - Готовая программа на .NET
